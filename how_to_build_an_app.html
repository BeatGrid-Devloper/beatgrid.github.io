What is the BG System:

BG Stands for BeatGrid. The BG System is the BeatGrid Custimer User Interface for everyone that uses a Service. Members can sighn up, Chat with users, cheack out events happining near them, and more. It is the comunity of Audio Enthusiasts for everyone in the world.

What is an App:

I am pritty sure you know what an app is. You hear App store, App Cloud, App building. We this is not to different. Lets say you want to build an app so that members on BeatGrid can sends photos like on snap chat. Well, you can definatly do that. All you have to do is build and desighn it. We hae tons of options from us intigrating it with our system, or runing it through an API (Comming Soon).

Types of APPs and Addons:

{ Host On }

The host on option is where instead of you buying resorces to run you app and contact us for conection, we run and intigrate it for you. You send us a link to the app on GitHub. For a more clear Documentation, Please see the wiki.

{ Off Campus (Curently Not Suported) }

Off Campus allows Custimers to use other members programs. For more information, please visit the Wiki here.

What Language is it in:

How do I build It:

You can easaly build it using programs and spoftware desighned to build the apps, or you can build it yourself. We have placed some templates on the GitHub Repo, so don't worry about problems. For a compleat how to, please read the WIKI.

Now What:

Once you have built your program, send it up to GitHub and send us a link. We will install it and test it befor releacing it. If your app passes inspection then we will let you know and releace it in our next update.
